# Hospital Landing Page

This project is a responsive hospital landing page built with **Gatsby.JS** and deployed on **Vercel**.  
It was developed as part of a React assignment to demonstrate skills in Gatsby, REST API integration, Axios, GitHub workflows, and deployment.


# Assignment Goal
The goal was to deploy a single page on Vercel with a **dynamic reviews section**.  
The reviews section consumes data from a REST API using Axios and renders it dynamically on the page.



# Technologies Used
- **Gatsby.JS** (React framework)
- **Axios** (for API calls)
- **REST API** (reviews data)
- **Figma** (UI design reference)
- **Git & GitHub** (version control)
- **Vercel** (deployment)
